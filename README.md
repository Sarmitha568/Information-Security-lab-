# Information-Security-lab-

## 1. Playfair Cipher

### Aim
To implement the Playfair Cipher algorithm for encryption and decryption using Java.

### Description
This program implements the Playfair Cipher by generating a 5×5 key matrix from a keyword, encrypting the plaintext, and decrypting the ciphertext back to the original message.

### Files
- `PlayfairCipher.docx` – Lab record

  ### Language
- Java
